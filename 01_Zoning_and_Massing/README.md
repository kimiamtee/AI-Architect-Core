# Autonomous Architectural Space Programming & Bubble Diagram Agent

An AI-powered computational architecture tool that takes raw, unstructured client design briefs, processes them using international residential standards, and automatically visualizes a high-end, professional spatial relationship (Bubble Diagram).

## 🚀 High-Income Skill Showcase
Instead of spending hours manually drafting space matrices and circulation flowcharts, this script automatically converts human-written text into an optimized, presentation-ready architectural diagram in under 2 seconds.

## 🛠 Features
1. **The Allocator (GPT-4o Matrix):** Reads the brief and precisely divides a specific square footage (e.g., 150 sqm) into functional rooms (Living, Kitchen, Master, etc.) ensuring total area compliance.
2. **The Visualizer (Matplotlib Vector Engine):** Maps geometric coordinates, scales bubble radii based on mathematical area calculations ($\sqrt{Area}$), and applies a curated studio-style architectural pastel palette.
3. **Circulation Graph:** Draws explicit node-to-node connectivity lines representing spatial adjacencies and flow.
